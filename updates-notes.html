Notes of the update to WebExtension

GKeep Panel now is Pinned Google Keep

Hi and thanks for using my addon all over these years!
(If you are a new user, then probably this message is not going to interest you too much.)

You will have noticed that with the last major Firefox update (Quantum) the addon stopped working.
This was because the Legacy addons are no longer supported and I had to port my code (rewrite almost everything from scratch) to the new WebExtensions technology.

With the new WebExtensions it can't be done everything what the Legacy addons could did.
Now the content inside the panels is loaded every time they are opened. I think with this behavior, the panel loses all the functionality it had in the past for my addon.
For this reason, I decided to use the pinned tabs. From now on, when the button is clicked, a pinned tab will open (or focus if you already have one). If you click again, you will return to the previous tab.
About the keyboard shortcut, once more, with WebExtensions the ability to configure it has been lost. So, I set the combination that I use: Alt + Shift + K. I hope that this can be configurable in the future, but all depends on Mozilla.

Well I hope this has not boring you and (I think) will be the only one time I'll bother you with a message like this. I'm sorry if I did and/or if you're disappointed with the change. This is the best approach I could find.
